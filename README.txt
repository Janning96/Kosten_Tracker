# Finance Import App — docs/ (GitHub Pages)
1) Kopiere den gesamten `docs/`-Ordner in dein Repo (Branch: main).
2) GitHub → Settings → Pages → **Source: main /docs** einstellen.
3) Öffne die bereitgestellte Pages-URL.

Lokaler Test: `docs/index.html` im Browser öffnen.

Hinweis: N26-PDF-Parser ist heuristisch und erwartet den Header **Description | Posting Date | Amount** im Text.
